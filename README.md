# Project Setup

## Run the Frontend

```bash
cd frontend
npm run dev
```

## Run the Backend

```bash
cd backend
uvicorn main:app --reload
```

## Installing Packages

### Install Backend Dependencies

1. Open your terminal
2. Navigate to the backend folder:

```bash
cd backend
```

3. Install the required packages:

```bash
pip install -r requirements.txt
```
